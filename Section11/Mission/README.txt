Réalisé par Martin Gyselinck

La principale difficulté fut de comprendre ce qu'on voulait, les consignes étant très peu claires...
Après, il s'en est suivi beaucoup d'essais pour comprendre comment gérer les noeuds, ce n'était pas vraiment facile...
Ensuite, ce fut au tour des tests... intestables, car faits en fonctions des objets qui leur serviront d'attributs.
Bref, la galère.

Mais sinon le code fonctionne, et ça c'est cool.

C'était la dernière mission, j'ai toujours pas compris comment on peut développer à deux des missions
dont le code fonctionne en tout ensemble, pas parallèlement.
